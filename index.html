About
Built by one person

Word Mining Game is an indie idle game made by PiffGames. What started as a simple concept grew into a full progression system with hundreds of ores, a live social layer, and a global community.

Everything — the engine, the balance, the backend — is built and maintained solo. If you enjoy the game, the best thing you can do is share it.

Major code assistance provided by Claude AI (Anthropic).
